## 1. 总体布局

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ Top Bar                                                                     │
│ [ Project Switcher ▼ ]   [ Search ]      [ Global Status ]   [ User Menu ]  │

│                       │                                      │               │
│ - Chats               │ 当前页面主体                         │ Session Info  │

```

### 说明

- **Top Bar**：全局层，始终可见
- **Left Sidebar**：当前项目导航
- **Main Content**：当前模块内容
- **Right Panel**：运行态辅助信息，可折叠

## 4. 项目内 Chat 列表

```text
┌──────────────────────────────┐
│ 项目：my-openclaw            │
│ [ + New Chat ]               │
├──────────────────────────────┤
│ Chats                        │
│ ● 项目化改造设计            │
│   PM Agent · 2 分钟前        │
│ ↻ 数据迁移方案              │
│   Backend Agent · 运行中     │
│ ! UI 原型说明               │
│   PM Agent · 等待输入        │
├──────────────────────────────┤
│ Agents                       │
│ - PM Agent                   │
│ - Backend Agent              │
│ - QA Agent                   │
└──────────────────────────────┘
```

### 状态含义

- `●` 最近活跃
- `↻` 正在运行
- `!` 等待用户输入
- `…` 后台处理中
- `✓` 最近任务已完成

## 15. MVP 页面范围

第一版最小可用建议只实现：

1. Project Switcher
2. 项目 Dashboard
3. 项目内 Chat 列表
4. Chat 详情页（显示项目名和默认 Agent）
5. Agents 页面
6. 新建 Project / 新建 Chat / 新建 Agent 弹窗

其他如 Memory / Activity / Files / Settings 可第二阶段补齐。
