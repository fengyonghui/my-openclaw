# my-openclaw ui

基于 `pm/` 和 `ux/` 文档整理出的前端实现骨架。

## 技术栈
- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## 已实现
- App Shell
- Project Dashboard
- Chat Detail
- 基础 Sidebar / Top Bar / Context Panel

## 启动
```bash
cd ui
npm install
npm run dev
```

## 当前定位
这是一个可运行的前端原型骨架，主要用于把项目化需求和 UX 设计落成页面结构。
后续可继续补：
- Agents 页面
- Files 页面
- Activity 页面
- 路由
- API 接入
- 状态管理
